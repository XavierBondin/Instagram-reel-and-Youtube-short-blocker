# 🎯 Lock In

Block Instagram Reels & YouTube Shorts on Web

A simple Chrome extension that helps you stay focused by removing distracting short-form content from:

* Instagram (Reels)
* YouTube (Shorts)
Find it at [
](https://chromewebstore.google.com/detail/hoeldclgkhhpbkkpgkepajadkfblfpkf?authuser=0&hl=en)
---

## 📦 Installation Guide

Follow these steps to install the extension manually:

1. **Download the files**

   * Clone or download this repository to your computer

2. **Open Chrome Extensions Page**

   * Go to: `chrome://extensions/`

3. **Enable Developer Mode**

   * Toggle **Developer mode** (top-right corner)

4. **Load the Extension**

   * Click **"Load unpacked"**
   * Select the folder containing the extension files

5. **Activate the Extension**

   * Make sure it is turned **ON** in your extensions list

---

## 🛠️ Notes

* Works only on **web versions** of Instagram and YouTube
* If the platforms update their layout, the extension may need updates
